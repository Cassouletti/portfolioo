    Introduction


    Dans ce projet, j'ai analysé un jeu de données sur l'absentéisme au travail en utilisant Python, afin de comprendre les facteurs qui influencent l'absentéisme et d'en tirer des insights pertinents.


    Avant de procéder l'analyse, un prétraitement minutieux des données a été effectué.


    Une fois les données nettoyées et préparées, j'ai réalisé une analyse préparatoire avec plusieurs visualisations pour explorer les relations entre les variables et identifier les tendances clés.

En régression, une variable muette est une variable qui prend une valeur binaire (0 ou 1) pour indiquer l'absence ou la présence d'un effet catégorique susceptible de modifier le résultat.

    La colonne Date

    Les dates dans la colonne sont actuellement des chaînes de caractères. Convertissons-les en format datetime.


                        
                        data['Date'] = pd.to_datetime(data['Date'], format="%d/%m/%Y")

    Je crée une colonne "mois" en vue d'une analyse avec des visualisations.


                        
                        months = [rows['Date'].month for index, rows in data.iterrows()]
                        name_months = {'Jan' : 1, 'Feb': 2, 'Mar': 3, 'April': 4, 'May': 5, 'June': 6, 
                                       'July': 7, 'Aug': 8, 'Sep': 9, 'Oct': 10, 'Nov': 11, 'Dec': 12}
                        def n_months(value, dictionary):
                            for x in dictionary:
                                if value == dictionary[x]:
                                    return x
                        data['Month'] = months
                        data['Month'] = data['Month'].apply(lambda x: n_months(x, name_months))










    En utilisant la bibliothèque Babel, je traduis les noms des mois dans la colonne "mois" en français, afin de rendre les analyses et visualisations plus pertinentes pour un public francophone.
